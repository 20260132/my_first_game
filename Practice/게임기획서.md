## 게임 기획서 (LMS)

## 게임 기본 정보
- 게임 제목 : Nova Blaster
- 장르 : 2D Shooter game
- 기반 템플릿 : Space Shooter.py

## 게임 규칙
- 목표: 주어진 시간 안에 점수를 얼마나 얻나
- 조작 방법: WASD로 움직여서 스페이스바로 적들을 처치해 점수를 얻는 방식이다

## 차별점
타이머 설정을 넣어서 게임이 흥미를 잃기 전 아쉽게 끝나 반복적으로 즐길 수 있게 함
밀쳐지는 아이템을 먹음으로써, 맞으면 바로 죽는 것을 없앰
중간에 블랙홀이 나옴으로써 빨려들어가는 신박한 요소를 넣음
랜덤 위치에서 생성되어 몇 초 후에 사라짐

## 난이도 설계
처음에는 죽이기만 하다가 밀치는 아이템을 먹고 나서는 최대한 뒤에 나오는
우주선과 같이 밀쳐 점수를 얻으려고 하는 점에서 난이도가 올라감
그리고 중간에 한번씩 블랙홀이 존재하면서 상대 우주선들이 빨려가지 않도록
빠르게 죽여야 함


## 무엇을 수정해야 되나?
1. 날라가는 총알 크기 수정(살짝 두껍게)
2. 총알 날라가는 속도 설정(제일 좋은 속도로)
3. 방향키 뿐 아니라 WASD로도 이동가능 하게 수정하기
4. 레벨 없애기
5. 우주선이 내려오는 형식이 아닌 랜덤 생성 후 5초 이후에 사라지기(시간은 최적 시간 찾기)

2. 밀치는 아이템 먹은 후 총에 맞으면 상대 우주선 적 조금씩 밀쳐지기
--> 여기서 생각할 것(뒤에 우주선이 있으면 같이 밀릴 것이냐, 닿은 애만 밀릴 것이냐? 아님 부딪쳐서 터지면서 사라지게 할 것이냐?)
3. 블랙홀 부분을 넣어서 나, 상대 우주선 둘 다 빨려들어가게 하는 요소 추가
4. 타이머 설정
